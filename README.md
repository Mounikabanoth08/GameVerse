# 🚀 GameVerse: A Gamified Technical Learning Arena

**GameVerse** is a professional full-stack web application designed to transform technical education into an interactive experience. Developed as part of the **Project-Based Learning (PBL)** curriculum, it allows students to master domains like SQL and Logic through competitive gameplay.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Backend:** Node.js, Express.js
- **Database:** SQLite3 (Relational Database)
- **Styling:** Futuristic "Nebula" Theme

## ✨ Key Features
- **Tiered Difficulty:** Choose between Easy, Medium, and Hard levels.
- **Game Modes:** Single Player, 2-Players, and Group modes.
- **Real-time Validation:** Instant feedback on quiz answers.
- **Data Persistence:** Scores and user progress are saved securely in an SQLite database.
- **Live Timer:** Tracks efficiency and problem-solving speed.
